### Running site locally

###### PREREQS
1. [Ruby](https://www.ruby-lang.org/en/downloads/) 1.9.3
2. Bundler <code>gem install bundler</code>

###### START SERVER
##### 
	rvm use 1.9.3
	sh start.sh

**http://localhost:4000**
